### 3. Configuration Git Automatique par Dossier

**Problème** : L'utilisateur a déplacé le projet Sabrina sous WSL (était sous Windows). La config Git locale était restée sur `ops@kossmonot.com`, causant un mélange des identités entre les 2 comptes GitHub (B2Big vs kossmonot-founder).

**Solution** : `includeIf` dans `~/.gitconfig`

```
[includeIf "gitdir:~/projets/sabrina/"]
    path = ~/.gitconfig-sabrina
[includeIf "gitdir:~/projets/Kossmonot_Web/"]
    path = ~/.gitconfig-kossmonot
```

| Dossier | Identité auto-appliquée |
|---------|------------------------|
| `~/projets/sabrina/` | `B2Big <ufcmjohan@gmail.com>` |
| `~/projets/Kossmonot_Web/` | `kossmonot-founder <ops@kossmonot.com>` |
