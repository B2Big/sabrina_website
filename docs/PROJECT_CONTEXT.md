# Contexte du Projet Sabrina - Coaching & Massage

## Vision Globale
Site web pour Sabrina proposant des prestations de coaching et de massage.
**Nouvelle Direction Artistique (Janvier 2026) :**
- **Thème :** "Guerrière / Amazone". Énergie sportive, force, vitalité.
- **Palette :** Nuances de rouges vibrants, design sombre ("Dark mode"), premium.
- **Ambiance :** Ultra moderne, animations "Wow effect", marketing agressif/efficace.

## Fonctionnalités Clés
1.  **Présentation des services :** Coaching et Massages.
2.  **Contact :** Formulaire de contact par email.
    -   *Feature demandée :* Notification SMS lors d'une soumission.
3.  **Supprimé :** Pas de module de réservation Calendly (gestion manuelle par Sabrina).
4.  **Localisation :** Pas d'adresse physique affichée.

## Stack Technique
-   **Framework :** Next.js 15 (App Router)
-   **Langage :** TypeScript
-   **Styling :** Tailwind CSS v4
-   **Animations :** Framer Motion
-   **Backend :** Server Actions pour le formulaire
