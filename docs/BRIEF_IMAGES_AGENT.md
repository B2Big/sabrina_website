## 📋 Contexte

Le site est une PWA Next.js avec une homepage qui contient :
- Un hero avec carrousel de 5 photos de Sabrina
- Une section Coaching avec cards de prix
- Une section Massage avec cards de prix
- Un photo marquee (défilement d'images)

**Ce qui manque** : Des visuels "vendeurs" qui créent de l'émotion avant même que le visiteur lise le texte.
